DESCRIPTION

If the radius within which the player can move becomes smaller, the squid game judges that it is a crisis situation and the background turns red.

Inspiration

Inspired by the games in the Netflix Original South Korean Drama "Squid Game" and professor Stavros's usage of window size interaction.

How to see the Green Light turning to Red Light

1. reload the page to make sure the game fits your window fully.

2. Move around the player

3. Exit the full screen
