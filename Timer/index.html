Custom Your Break-Time

:

:

This site is a free space where all users can take a break.
I hope you set a time that you need and rest as deeply as possible during that time.
You can write down the desired time by pressing the number, or you can easily set it by pressing the up/down arrow buttons.
Press the play button to start the timer and overwind button to reset.
The flowers in the background are created infinitely, so you can press the flower to stop it if it bothers you.
However, it has not yet advanced to the stage where flowers are created again.
